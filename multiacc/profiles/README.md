M-profiles

Метод мультиаккаунтности, основанный на профилях CLI

Как юзать:
Клонировать/вгетнуть в папку с CLI, дать файлам init.sh и start.sh права на исполнение (`chmod 777 init.sh start.sh`), запустить init.sh, через него настроить CLI (По умолчанию десять ботов, больше достигается ctrl_c_v профилей и скриптов).
После этого `nano start.sh`, вбиваем твинкам их замки после --castle, выходим и `./start.sh ваш_ник_с_регистром_но_без_@ голд_который_оставляем_после_доната название_группы_с_ботами`, у кого ещё нет группы - настоятельно рекомендую создать.
Всё, идём попивать чаёк
